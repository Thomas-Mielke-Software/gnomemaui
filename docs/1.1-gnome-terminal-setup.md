# GNOME Terminal setup for GNOME MAUI .NET development environment

> [!NOTE]
> This documentation depends on the [# Development Environment Setup for GNOME MAUI .NET on Linux](/docs/1-devenv.md) documentation.

This document describes how to set up GNOME Terminal to work with the GNOME MAUI .NET development environment.

In the GNOME Terminal menu, open the "Preferences" menu item.

![Preferences window](/assets/gnome-terminal/preferences.png)

Clone the existing profile (e.g., "Unnamed") and name it "GNOME MAUI".

![Clone profile](/assets/gnome-terminal/clone.png)

On the Command tab, check the "Run a custom command instead of my shell" option and enter the following:

```bash
bash -c "cd $GNOMEMAUI && exec bash --rcfile $GNOMEMAUI/.vscode/.linux/.bashrc"
```

![Set custom command](/assets/gnome-terminal/command.png)

In `.vscode/.linux/install.env` change PROJECT_TERMINAL="gnome-terminal" and set the GNOME_TERMINAL_ID var to the GUID of your GNOME Terminal profile, which is displayed in the lower-right corner of the profile preferences.

Close the Preferences window.

From now on, you can start the GNOME MAUI .NET development environment by selecting the "GNOME MAUI" profile in GNOME Terminal.

![Select GNOME MAUI profile](/assets/gnome-terminal/profile.png)

Install NodeJS:

![Install NodeJS](/assets/gnome-terminal/nodejs.png)

You can now continue with the [patch](/docs/2-patch.md) documentation.
